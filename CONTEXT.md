# CONTEXT

Diagram Lab 的領域詞彙表。

## 詞彙

### 節點 (Node)

圖面上的圓形實體,持有**名稱**、**數值**與位置。名稱以字母接序自動命名(A、B、C…)。

### 數值 (Value)

節點持有的非負整數,不可為負。數值只透過**移轉線**在節點間移動。

### 移轉線 (Transfer Line)

兩個節點間的有向關聯,自**來源節點**每秒移轉 1 點數值到**目標節點**。

規則:
- 來源節點數值為 0 時,該移轉線**暫停**;來源再獲得數值即自動恢復。
- 同一秒內多條移轉線依**建立順序**逐條結算,來源不足時該條該秒略過。
- 禁止自連(來源=目標)與重複同向線;反向並存(A→B 與 B→A)允許。
